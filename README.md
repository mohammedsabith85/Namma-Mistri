# Namma Mistri (ನಮ್ಮ ಮಿಸ್ತ್ರಿ) 🏗️

**Namma Mistri** is an offline-first Android application designed to empower local construction contractors and site managers. It replaces traditional paper-based site management with a digital toolkit that works entirely without internet, featuring seamless English-Kannada bilingual support.

## 📌 Problem Statement
Local construction managers often struggle with data loss from physical notebooks, inaccurate material estimations leading to waste, and a lack of accessible tools that support regional languages. Namma Mistri bridges this gap by providing a localized, digital solution for grassroots construction professionals.

## ✨ Key Features
* **Multi-Site Tracking:** Organize and monitor multiple construction plots simultaneously.
* **Smart Material Calculator:** Instant on-site estimations for bricks, cement, sand, and concrete.
* **Labor Management:** Track workforce attendance and daily wage distribution across sites.
* **Bilingual Support:** Instant toggle between English and Kannada for better accessibility.
* **100% Offline:** All data is stored securely on the device, requiring no internet connectivity.

## 🛠️ Technologies Used
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Material Design 3)
* **Architecture:** MVVM (Model-View-ViewModel)
* **Local Storage:** SharedPreferences & Gson
* **Version Control:** Git & GitHub
